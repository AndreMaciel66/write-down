# write-down

一个使用React编写的写作平台。

## Demo

[Try it!](https://nevenleung.github.io/write-down/)

## 功能列表

### 文章系统

- [X] 文章编辑
  - [x] 滚动同步的开启与关闭
  - [x] 编辑与预览模式切换
- [ ] 文章管理
- [ ] 文章导入与导出

## Script

Run

```
npm run start
```

Build

```
npm run build
```
